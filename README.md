1.  Clone the repository: git clone <git@github.com>:mmelei/M_cubed_repo

2.  Navigate to the correct directory in terminal (cd .../M_cubed_repo) 

3.  Run the “make” command

Docker image (irene5297/final) to one that would be accessible on your account. After that you can run "make".

- If you run "make" in the terminal and subsequently make alterations to the code, run "make clean" 
before running "make" again.
